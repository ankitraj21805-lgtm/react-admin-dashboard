# SOUL HQ

Premium responsive MadOut 2 clan website for SOUL Syndicate.

- Website: SOUL HQ
- Alliance tag: SYN
- Tagline: Respect • Loyalty • Skill
- Stack: Next.js, TypeScript, Tailwind CSS, Supabase-ready, Vercel-ready
- Demo admin/tryout storage uses browser localStorage so it works live without secrets.

## Routes
Home, About, Members, Join, Events, Alliance, Gallery, Rules, Login, Admin.

## Run
```bash
npm install
npm run dev
```

## Optional Supabase env
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_project_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
